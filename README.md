# Cloudflare-error-page-HTML

## 项目介绍 (Project Introduction)

### 中文
这是一个基于 [donlon/cloudflare-error-page](https://github.com/donlon/cloudflare-error-page) 构建的 Cloudflare 错误页面 HTML 项目。该页面具有以下特点：
- 动态显示 UTC 时间
- 自动获取用户的 IP 地址
- 生成随机的 Ray ID

### English
This is an HTML project for Cloudflare error pages built on [donlon/cloudflare-error-page](https://github.com/donlon/cloudflare-error-page). The page features:
- Dynamic display of UTC time
- Automatic acquisition of the user's IP address
- Generation of random Ray IDs
